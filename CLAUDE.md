## Claude への指示

- 各サブディレクトリの CLAUDE.md にその部門固有のルールを記載している
- 文書を作成する際は必ず `context/` の関連ファイルを参照すること
- 曖昧な指示を受けた場合は、三幸商事の鉄鋼商社としての業務文脈で解釈すること
- 金額・数量・納期を含む文書には必ず確認を促すコメントを添えること
- タスク管理・進捗確認の依頼があれば `context/asana.md` を参照し、Asana API を呼び出して対応すること
- スケジュール・予定確認の依頼があれば `context/google_calendar.md` を参照し、iCal URL からカレンダーデータを取得して対応すること
- メール確認・検索の依頼があれば `context/gmail.md` を参照し、IMAP で Gmail に接続して対応すること
