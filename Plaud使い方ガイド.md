## 新しい録音を追加するとき

Plaudで録音 → アプリが自動同期 → 以下のコマンドを実行：

```bash
node "\\192.168.1.126\本社\営業\営業2部\個人フォルダ\竹村\カーソル\三幸商事株式会社\plaud_sync.mjs"
```

※差分のみ同期（既存ファイルはスキップ）

## トークンの期限

Plaud APIトークンの有効期限: **2026年9月3日**

期限切れになったら：
1. Chrome で [web.plaud.ai](https://web.plaud.ai) を開く
2. DevTools → Application → Local Storage → https://web.plaud.ai
3. `tokenstr` の値をコピー
4. `C:\Users\FONE\.plaud\config.json` の `token.accessToken` を更新
