## 見送り3案の理由（デビルズアドボケイトの指摘）

### D. スクラップリサイクル
- 粗利率2〜3%（現行鉄鋼商社6.9%を**下回る**）
- 大阪でのヤード確保・トラック・クレーン等の異業種設備が必要
- 鉄スクラップ相場変動リスクを二重に抱える
- 仕入先メーカーが「廃品業者と同列に見られたくない」と関係悪化リスク

### E. EV・特殊鋼材専門商社化
- ハイテン鋼板・電磁鋼板はメーカーが販売先を厳格管理 → 後発の中小商社に特約は取れない
- 阪和興業・メタルワン・JFE商事と正面対決は無謀
- 粗利率は汎用材と大差なく（5〜8%）、在庫リスクだけが膨らむ
- **結論**: 新規事業としてではなく、日本製鉄・JFEとの関係深化（89期既存方針）の中で自然に取扱比率を上げる

### F. IPライセンス × 完成品缶バッジ
- 版権交渉相手はバンダイナムコ・集英社・角川 → 三幸商事の名前でテーブルに着けるか疑問
- **致命的**: 既存のB2B顧客（funbox×CAN-缶・ベック等）のビジネスに直接競合する
- IP人気の突然の終焉で在庫がデッドストックになるリスク

## 推奨事業案A：缶バッジマシン＋パーツの海外代理店展開

### 事業の本質

**国内でやっていることをそのまま海外に持ち出す。**

三幸商事は国内でfunbox×CAN-缶・ベック・サザビー等に缶バッジマシン＋パーツを供給し、マシン購入者をパーツリピーターに育成している。この**「マシンを入口にしたパーツのサブスク的収益」フライホイール**は89期計画で国内強化方針が既に確定している。

海外展開でやることは単純だ：**信頼できる現地代理店を見つけ、国内モデルを現地で再現させる。**

自社でECを運営する・OEM製造を受ける・完成品を直販する——いずれも不要。三幸は「製品を卸す」だけでよい。現地事情・顧客開拓・アフターサポートは代理店に任せる。

### フライホイールの仕組み（国内→海外）

```
【国内での実証モデル（再確認）】
  三幸プロダクツ（製造）
      ↓ パーツ供給
  代理店（funbox×CAN-缶・ベック等）
      ↓ マシン販売 + パーツ卸
  エンドユーザー（個人・同人作家・グッズ業者）
      ↓ マシンを使い続ける限りパーツを消費
  継続的パーツ需要 → 三幸への定期発注

【海外で再現するモデル】
  三幸プロダクツ（製造）
      ↓ パーツ + マシン供給
  現地代理店（韓国・台湾・北米等）
      ↓ マシン販売 + パーツ卸
  現地エンドユーザー（K-pop推し活・アニメファン・グッズ業者）
      ↓ 継続的パーツ消費
  代理店からの定期発注 → 三幸への安定売上
```

### ターゲット市場と優先順位

| 市場 | 理由 | 優先度 |
|------|------|--------|
| **台湾** | 日本の缶バッジ文化が最も浸透。日本語対応可能な業者多数。地理的近接 | **最優先** |
| **韓国** | K-pop推し活文化でアクリルスタンド・缶バッジ需要が爆発的。競合も強いが市場も大きい | **優先** |
| **北米（米国）** | Anime Expoを中心としたアニメグッズ市場。英語対応が必要。物流コスト大 | 中期 |
| **東南アジア** | タイ・インドネシアでアニメ文化拡大中。89期は調査のみ | 長期 |

### 代理店選定：何を見るか

海外展開の成否は**最初の代理店1社**で決まる。選定基準を厳密に設けること。

**必須条件（Mustの条件）：**

| 条件 | 確認方法 |
|------|---------|
| 缶バッジ・グッズ関連の既存販売実績 | カタログ・取引先リスト・SNS確認 |
| 財務的安定性（在庫を持てる資金力） | 与信調査（帝国データバンク現地版・D&B等） |
| 現地顧客ネットワーク（ファンイベント・グッズ店等） | 代表者へのヒアリング・現地の評判 |
| 日本語または英語でのコミュニケーション能力 | 商談時に確認 |
| 競合製品（中国製格安マシン）を取り扱っていない | 商談・現地調査 |

**加点条件（Betterの条件）：**
- 同人誌即売会・アニメイベントの運営またはスポンサー実績
- 自社ECサイト・SNS（フォロワー数）保有
- 過去に日本製品の輸入・販売経験あり
- アフターサービス・修理対応ができる技術スタッフ保有

**即排除（Killの条件）：**
- 中国製廉価マシン・パーツの取扱がある（価格競争に引きずり込まれる）
- 同カテゴリーの複数ブランドを並売している（熱量が分散する）
- 担当者が頻繁に変わる・組織が不安定

### 代理店の見つけ方

**ステップ1：展示会での直接発掘（最重要）**

| 展示会 | 開催地 | 目的 |
|--------|--------|------|
| 台北国際書展・CWT（Comic World Taiwan） | 台湾・台北 | 台湾代理店候補の発掘 |
| KCON Japan / KCON USA | 大阪・ロサンゼルス | 韓国・北米向け推し活業者との接触 |
| Anime Expo | ロサンゼルス | 北米市場の業者・卸商との商談 |
| Wonder Festival / コミックマーケット | 国内 | 海外バイヤーが来日するタイミングを狙う |

> **ポイント**: 展示会は「三幸が出展」するより、まず「視察＋名刺交換＋商談」で十分。出展は2年目以降でよい。

**ステップ2：JETROの活用**

JETROは無料で現地の業者紹介・市場調査をサポートする公的機関。

- JETRO台北事務所・ソウル事務所に「缶バッジマシン・パーツの現地代理店を探したい」と問い合わせ
- 「新輸出大国コンソーシアム」や「海外展開支援プログラム」を活用することで費用を抑えながら代理店候補リストを入手可能
- 大阪のJETROアドバイザー面談（無料）から始める

**ステップ3：既存顧客の海外コネクション活用**

- funbox×CAN-缶・ベック等の国内顧客が海外バイヤーと接点を持っている可能性が高い
- 「海外の知人でこのビジネスに興味を持ちそうな方はいますか？」と聞くだけで候補が出てくることがある
- インバウンド来店・海外ECの購入者データから有力業者を特定する方法もある

**ステップ4：SNS・LinkedIn調査**

- 台湾・韓国のInstagram・X（Twitter）で「缶バッジ」「カンバッジ」「#canbadge」を検索
- 大量フォロワーを持ち、缶バッジグッズを頻繁に投稿しているアカウントはビジネス化済みの可能性が高い
- LinkedInで「badge maker distributor」「k-pop goods」等で検索、流通業者・輸入業者に絞る

### 代理店契約の構造

不誠実な代理店リスクを避けるため、**契約設計が命**。

```
【推奨する代理店契約の骨格】

1. テリトリー権：国単位での専売権（独占は慎重に。まず非独占から）
   → 実績を確認した上で1〜2年後に独占化を検討

2. 最低購入コミット：年間〇〇万円分の発注義務
   → 守れない場合はテリトリー権を失効させる条項を設ける

3. 禁止事項：競合製品（中国製格安マシン・パーツ）の同時取扱禁止

4. 再販価格：最低再販価格（MAP）を設定。ダンピング防止

5. サポート義務：エンドユーザーへの機械操作指導・初期サポートを代理店が担当
   → 三幸は代理店への技術研修を年1〜2回提供

6. 報告義務：月次の販売実績・在庫レポートの提出

7. 契約期間：初回1年間（更新制）。問題あれば更新しない形で柔軟に対応
```

> **注意点**: 独占契約を最初から与えると、代理店が動かなくてもテリトリーを押さえられ動けなくなるリスクがある。まず非独占で始め、実績が出てから独占化が安全。

### 収益モデル

| 収益源 | 単価イメージ | 粗利率 |
|--------|------------|--------|
| 缶バッジマシン（初回） | 数万〜数十万円/台 | 20〜35% |
| パーツ（継続）| ロット単位で定期受注 | 30〜45% |
| 消耗品（フィルム・鉄板等） | 少額・高頻度 | 40〜55% |

**フライホイール効果**: マシン1台の販売が、その後数年間のパーツ定期収入を生む。マシンは「先行投資」、パーツが「安定収益」。

## 推奨事業案B：缶バッジ隣接 金属加工受託の拡張

### なぜこの事業か

三幸プロダクツのプレス設備は缶バッジで稼働率65〜70%。**残り30〜35%の余力を活かす**のが最も資産効率の高い選択肢。  
批判的思考の指摘通り、医療機器（ISO13485・薬機法）は89期の参入対象から外す。まず**電子機器・食品缶隣接部品**から入り、認証を積みながら医療へ段階的に移行する。

### 4エージェントの評価

| 観点 | 評価 |
|------|------|
| 市場調査 | 食品缶部品市場1,500〜2,000億円。プレス技術転用で「ゼロコスト立ち上げに最も近い候補」 |
| 財務 | 粗利率20〜25%（商社の3倍）。設備増強込みで4〜5年投資回収 |
| 事業企画 | 試作→量産フローが三幸プロダクツ×三幸商事の一気通貫で差別化 |
| デビルズアドボケイト | 「既存製缶ユーザー顧客へのクロスセルから始めれば新規開拓コストゼロ」 |

### ビジネスモデル

```
収益構造:
  ① 試作受注（10〜50万円/件）→ 顧客との関係構築の入口
  ② 量産受注（100〜500万円/件）→ 本体収益
  ③ 「素材（三幸商事）+ 加工（三幸プロダクツ）」セット受注 → 粗利2重取り

優先ターゲット順:
  1位: 既存の缶ユーザー顧客（営業1部）の缶エンド・缶蓋部品
     → 既存信頼関係あり、新規開拓コストゼロ
  2位: 関西圏の電子機器メーカー（シールド板・ブラケット等）
     → ISO9001取得後に本格参入
  3位: 医療機器（介護ベッド・器具筐体）
     → 90期以降、認証取得後
```

## 推奨事業案C：鋼材調達デジタル化サービス（規模縮小・再定義版）

### 批判的思考による根本的な再定義

デビルズアドボケイトの指摘：**「SaaSはソフトウェア会社が作るもの。鉄鋼商社がフルスクラッチSaaSを内製するのは非現実的」**

→ 財務エージェントが見抜いた本質：**「SaaSの月額収益より、SaaS経由で発生する鋼材取引の方がはるかに大きい収益源」**

この2つを統合した再定義：

> **SaaSを作るのではなく、「発注ポータル（軽量デジタル発注窓口）」を既存顧客向けに提供し、三幸商事への発注を囲い込む手段として使う。**

外部SIerに本格SaaSを発注する（2,000〜4,000万円）のではなく、kintone・Notionベース等の**ローコードツールで60〜120万円**で作り、既存顧客10〜15社に無償提供から始める。

### 再定義後のビジネスモデル

```
【本来の目的の再確認】
  ゴール: 既存顧客の鋼材発注を三幸商事に集中させ、他社への流出を防ぐ
  手段: 発注が「楽になる」デジタル窓口を無償で提供
  収益: 発注ポータル経由の鋼材取引増分（既存顧客の購買シェアを70%→90%へ）

【収益計算（現実的な試算）】
  既存顧客20社が月平均500万円の発注を三幸に集中
  → 追加取引額: 月間1,000万円（シェア20%増加分）
  → 粗利（7%）: 月間70万円 → 年間840万円
  → ポータル構築費120万円は1.7ヶ月で回収

【将来の発展（90期〜）】
  新規顧客へのSaaS有償提供（月額3〜5万円）
  外部SaaSパートナーとの協業（開発リスクを持たない）
```

### 89期の行動
