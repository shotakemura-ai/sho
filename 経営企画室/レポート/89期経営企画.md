# 三幸商事株式会社 第89期 経営企画
**対象期間：2026年6月〜2027年5月**
**策定：経営企画室 / 承認：竹村 翔**
**策定日：2026年3月19日**

---

## エグゼクティブサマリー

89期は「4本柱の強化と拡大」を基本方針とする。

国内鉄鋼需要は構造的縮小が続くが、競合の淘汰が進む今こそ**メーカーとの関係深化**と**付加価値型営業への転換**でシェアを取りに行く局面である。缶バッジ事業は縮小ではなく**国内シェア拡大＋海外代理店モデルによるマシン・パーツ展開**で成長を目指す。営業2部は缶バッジ×薄板の2本柱、営業3部は直接貿易の強化、三幸プロダクツは営業2部と一体となり缶バッジ事業の供給体制を担う。

日本製鉄・JFEスチール・東洋鋼鈑・東京製鐵の4メーカーとの懇意な関係構築を全社の最優先課題として位置づける。

## 1. 外部環境分析

### 1-1. 鉄鋼業界の現状と89期見通し

| 項目 | 現状 | 89期見通し |
|------|------|-----------|
| 国内鉄鋼消費量 | 約5,500万トン（2025年） | 約5,400〜5,500万トン（横ばい〜微減） |
| 薄板系（冷延・TFS）需要 | 構造的縮小（年▲1〜3%） | 縮小継続。製缶向けは特に軟調 |
| 鋼材価格 | 高止まり後に軟化傾向 | 熱延 100,000〜115,000円/トン水準 |
| 競合環境 | 中小問屋の廃業・再編が加速 | 淘汰局面→シェア獲得のチャンス |
| 中国材流入リスク | 輸出攻勢継続（2024年：1億トン超） | 欧米規制強化→アジア迂回リスクあり |
| 為替 | 2024年：1ドル＝155〜160円 | 2026〜2027年：140〜150円方向へ是正 |

**競合他社の廃業・縮小が続く今が攻めどき。** 価格競争に引きずられず、専門性と信頼でシェアを取りに行く。

### 1-2. 重要メーカー動向

| メーカー | 特徴・注目点 |
|----------|-------------|
| **日本製鉄** | 国内最大。TFS・冷延・高張力鋼板の主要供給源。直販強化の動きあり。懇意な関係があれば情報優先提供・安定供給が期待できる |
| **JFEスチール** | 関西地区での代理店制度見直しの動き（2024〜2025年）。関係強化が急務 |
| **東洋鋼鈑** | TFS（ティンフリースチール）の専業メーカー。製缶用途で三幸商事との親和性が高い |
| **東京製鐵** | 電炉メーカー。薄板・形鋼の幅広い品種。コスト競争力が強み |

### 1-3. 缶バッジ市場の環境

**国内市場:**
- 「推し活」文化の定着により、缶バッジは安価・汎用なグッズとして安定需要が継続
- 2024年ピーク後の在庫調整（2025年）は一巡しつつある
- 中国製廉価パーツの流入が価格競争を引き起こしているが、品質・対応力で差別化可能
- ペットフード缶需要は年率+5〜8%成長セグメント

**海外市場（プラスアルファ）:**
- 日本の缶バッジ・推し活文化は東アジア・北米のアニメファンに支持
- Made in Japan の高品質パーツへの需要が海外ECで開拓余地あり
- インバウンド観光客向けのグッズ消費も継続

## 2. 89期 経営方針

### 基本方針

> **「4本柱の強化と拡大」**
> メーカーとの関係を深め、缶バッジ×薄板×貿易×加工の複合力で攻める。

### 方針の核心

1. **日本製鉄・JFEスチール・東洋鋼鈑・東京製鐵との懇意な関係構築**（全社最優先）
2. **缶バッジ事業：国内シェア拡大＋海外プラスアルファ**（縮小しない・拡大を目指す）
3. **営業2部：缶バッジ事業＋薄板事業の2本柱で成長**
4. **営業3部：直接貿易を強化**
5. **三幸プロダクツ：営業2部と一体で缶バッジ事業を支える**

## 3. 事業部別戦略

### 営業1部（缶ユーザー向け）

**現状評価:** 製缶メーカーとの長年の信頼関係と専門知識が強み。主力市場の食品缶が構造的縮小フェーズ。

**戦略方向性：「専門性の深化」で縮小市場での絶対的ポジションを確立**

食品缶の縮小を、ペットフード缶・防災備蓄缶・輸出缶詰向けへの横展開で補う。

#### 重点施策

**① 成長ニッチへの転換（ペットフード缶・防災缶詰）**
- ペットフード缶メーカー（89期中に5社アプローチ）を新規開拓
- 防災意識向上の追い風を活かし、自治体・食品メーカーの備蓄缶需要を取り込む
- 缶詰輸出メーカー（アジア向け）への素材供給を開拓

**② 東洋鋼鈑との関係強化で TFS 安定供給体制を構築**
- TFS（ティンフリースチール）の主要サプライヤーである東洋鋼鈑と定期戦略対話を設置
- 在庫確保力・ロット調整力を武器に「頼れる中間商社」としてのポジションを確立
- 競合中小問屋の廃業後の顧客を積極的に吸収する

**③ 三幸プロダクツとの「素材＋加工」セット受注**
- 缶素材の供給だけでなく、三幸プロダクツの缶部品加工（缶㏤天板等）をセットで提案
- 89期中に製缶メーカー3社以上でセット受注を実現

#### KPI

| 指標 | 89期目標 |
|------|---------|
| ペットフード缶・防災缶詰向け新規顧客 | 3社以上 |
| セット受注（素材＋加工）件数 | 3件以上 |
| 競合廃業顧客の吸収 | 5社以上 |
| 粗利率 | 前期比横ばい以上 |

### 営業2部（缶バッジ事業＋薄板事業）

**89期の重点：缶バッジ事業と薄板事業の2本柱で成長する**

#### 缶バッジ事業

**現状:** 三幸プロダクツと連携し、缶バッジパーツを国内主要プレイヤー（funbox×CAN-缶、ベック等）に供給。2024年好調→2025年在庫調整局面を経て、2026年は再拡大フェーズへ。

**方向性：国内シェア最大化＋海外代理店モデルによるマシン＋パーツ展開**

**① 国内主要顧客の深耕とシェア拡大**
- funbox×CAN-缶・ベック・サザビー等の主要バイヤーとの年間供給枠を正式合意（MOU締結）
- 競合パーツサプライヤーの弱点（品質・納期・在庫対応）を突いてシェアを奪取
- 未取引の国内缶バッジパーツ流通業者・グッズメーカーをリストアップし、89期中に10社以上にアプローチ

**② 海外展開：信頼できる現地代理店によるマシン＋パーツ展開**

国内モデル（マシンを入口にしたパーツのリピート購買フライホイール）を海外で再現する。  
自社ECや完成品OEMは行わず、**信頼できる現地代理店1社を見つけて国内モデルを現地に移植**することに集中する。

- **優先市場**: 台湾（最優先）→ 韓国 → 北米（中期）
- **代理店選定基準**: 缶バッジ・グッズ関連の既存販売実績 / 財務的安定性（在庫保有力）/ 現地イベント・ファンコミュニティとの接点 / 競合（中国製廉価品）の非取扱
- **代理店の探し方**: ①国内既存顧客（funbox×CAN-缶・ベック等）の海外コネクションを当たる ②JETROの台北・ソウル事務所へ無料相談 ③台湾・韓国の展示会（CWT・KCON等）を視察し直接商談
- **契約設計**: 最初は非独占契約から開始。6〜12ヶ月の実績確認後に独占化を検討。最低購入コミット・競合品取扱禁止・MAP（最低再販価格）を必ず設定
- **89期上半期中に台湾または韓国で代理店候補1社とMOU締結**を目標

**③ 缶バッジマシンの戦略的活用（国内＋海外共通）**
- 缶バッジマシンを「パーツの入口」と位置づけ、マシン購入者をパーツリピーターに育成する仕組みを強化
- 国内：マシン販売後のパーツ定期購入プログラム（サブスク的な継続購買）を設計
- 海外：代理店経由でマシンを普及させ、代理店からのパーツ定期発注を安定収益源とする

#### 薄板事業

**方向性：付加価値型営業＋品種選択と集中**

**④ シャーリング加工の提案型営業への転換**
- 「頼まれたら加工する」受け身型から、顧客の図面を先読みした提案型営業に転換
- 加工付き案件の粗利総額を前期比120%に引き上げ

**⑤ 日本製鉄・JFEスチール・東京製鐵との関係強化で価格競争力と情報優位を確保**
- 各社担当窓口との定期情報交換（月1回）を仕組み化
- 最新鋼種情報・新製品情報を顧客への付加価値として活用

**⑥ 品種選択と集中（利益率の低い取引の整理）**
- 粗利率2%以下の品種・取引先を89期開始前（2026年5月）に棚卸し
- 空いたリソースを缶バッジ事業と高粗利薄板品種に再配分

#### KPI

| 指標 | 89期目標 |
|------|---------|
| 缶バッジ 国内主要バイヤーとのMOU締結 | 3社以上（89期上半期中） |
| 缶バッジ 新規取引先 | 10社以上 |
| 缶バッジ 海外代理店候補とのMOU締結 | 台湾または韓国で1社以上（89期上半期中） |
| 缶バッジ 海外 代理店経由マシン初回販売 | 89期Q3中（2026年12月〜2027年2月） |
| シャーリング加工付き案件 粗利総額 | 前期比120% |
| 薄板 品種別収益マップ作成 | 2026年5月末 完成 |

### 営業3部（直接貿易）

**89期の重点：直接貿易を強化する**

**現状評価:** 貿易実務能力（通関・輸送・与信管理）は三幸商事の希少な強み。直接貿易（自己リスクを取った輸入・輸出）の拡大で収益を伸ばす。

**方向性：直接貿易の拡大と収益基盤の確立**

#### 重点施策

**① 輸入直接貿易の拡大（国内顧客への安価・多様な素材供給）**
- 韓国（POSCO）・台湾（中鋼）等からの冷延・薄板の輸入を拡大
- 円高方向への是正局面（1ドル=140〜145円想定）を追い風に採算確保
- 国内顧客（営業1部・2部の既存顧客）への輸入材提案を強化し、メリットを訴求
- 89期上半期中に新規輸入先（サプライヤー）を2社以上開拓

**② 輸出直接貿易の拡大（日本産高品質鋼材を海外へ）**
- 東南アジア（ベトナム・インドネシア・タイ）向けの輸出を拡大
- 日本製鉄・JFEスチール・東京製鐵の高品質材の付加価値を訴求し、単純な価格競争を回避
- 89期中に東南アジア新規バイヤー2社以上との取引を開始
- 年間目標：直接貿易 新規売上2億円

**③ 直接貿易の採算管理の強化（リスク管理と収益確保の両立）**
- 為替予約・オープンポジション管理を徹底（大口案件は受注時に70%以上ヘッジ）
- 案件ごとに「為替・物流コスト・与信コスト込みで粗利X%以上」の採算基準を明文化
- 採算割れ案件は受注前に経営企画室と協議するルートを設置

**④ メーカー4社との関係を貿易チャネルとして活用**
- 日本製鉄・JFEスチール等のメーカーが輸出したい商材を直接貿易でさばく体制を構築
- メーカー側のニーズ（輸出先・数量）を把握し、三幸商事が受け皿となる関係を作る

#### KPI

| 指標 | 89期目標 |
|------|---------|
| 新規輸入先（海外サプライヤー）開拓 | 2社以上 |
| 東南アジア向け新規バイヤー | 2社以上 |
| 直接貿易 新規売上 | 2億円 |
| 為替ヘッジ遵守率（大口案件） | 100% |

## 5. 全社横断施策

### シナジー活用

**「鉄鋼素材 → 加工 → 最終顧客」バリューチェーン一体提案**
- 営業1部（缶素材）× 三幸プロダクツ（缶部品加工）のセット受注を拡大
- 製缶メーカーに「鋼板仕入れから缶部品製造まで一社完結」を提案

**「缶バッジマシン＋パーツ × 現地代理店 × 直接貿易」でグローバル展開**
- 国内で実証済みの「マシンを入口にしたパーツのリピート購買」フライホイールを海外に移植
- 自社EC・OEM製造は行わず、**信頼できる現地代理店（台湾・韓国を優先）がマシンを販売 → パーツを三幸から定期調達**するモデルを構築
- 現地代理店の発掘はJETRO活用 + 既存国内顧客のコネクション + 現地展示会視察の三本立て
- 代理店への物流・通関は営業3部の直接貿易ルートを活用
- 営業2部（代理店管理・製品企画）× 三幸プロダクツ（製造・供給）× 営業3部（輸出通関・物流）の三部門連携

### 競合との差別化

- **情報提供力とスピード**：メーカー最新情報を月1回メールマガジン形式で主要顧客に配信
- **見積回答：翌営業日24時間以内**を全営業で統一
- 大手商社が対応しない「中小顧客・小ロット・短納期」に特化したサービスレベル
