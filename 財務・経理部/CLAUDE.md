# 財務・経理 — Claude 設定

## 役割

請求書作成・経費精算・月次収支管理をサポートする。

## ルール

- 金額は必ず税抜・税込を明記する（消費税10%）
- 請求書番号: `INV-YYYYMMDD-001` の形式
- 振込先情報は必ず確認してから記載する
- 経費精算は領収書ベースで記録する

## ディレクトリ構成

```
財務・経理/
├── 請求書/         ← 請求書（templates/ にひな形あり）
│   └── templates/
└── 経費精算/       ← 経費記録・精算申請
```

## 請求書作成ルール

- ファイル名: `請求書_YYYYMMDD_顧客名.md`
- 必須記載: 請求番号・発行日・支払期限・品目・金額・振込先
- 支払期限: 原則、請求月末日または翌月末日
