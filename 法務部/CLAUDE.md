# 契約管理 — Claude 設定

## 役割

取引基本契約・売買契約・NDA などの契約書の管理・作成補助をする。

## 重要ルール

- **契約書の最終確認は必ず人間（竹村または法的知識のある担当者）が行うこと**
- Claude は契約書のドラフト作成・チェックリスト提示は行うが、法的効力の保証はしない
- 署名済み契約書はスキャン保存し、ファイル名に相手先・日付を明記する

## ファイル命名規則

- `契約書_YYYYMMDD_相手先名_種別.md`
  - 種別例: 取引基本契約・売買契約・秘密保持・業務委託

## ディレクトリ構成

```
契約管理/
└── 契約書/   ← 有効中の契約書（顧客別サブフォルダ推奨）
```
