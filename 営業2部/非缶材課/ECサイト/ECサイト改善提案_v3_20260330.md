## 前提：ターゲットと現状把握

### ターゲット顧客
- **一定規模の法人**（個人・小規模事業者は対象外）
- 最小ロット = 三幸プロダクツの段ボール1箱単位（57mmなら2,000個）

### 現在の注文構成
```
EC注文                30%  ← 現状
メール文面だけ        ~15% ← 慣れた顧客。無理に変えなくていい
自社フォーマット注文書 ~10% ← 移行不可。受け入れる
メール注文書          ~45% ← EC移行ターゲット
```
**目標：EC比率 30% → 60〜65%**（メール注文書層をECの掛け払いに移行）

### 現在の決済方法
- 事前振込（全顧客）
- 自社審査の掛け払い（法人のみ。社内審査済み）
- クレカ：過去に対応していたが廃止（在庫切れ時の返金が煩雑だったため）
  → **57mmの生産能力拡張により在庫切れがほぼ発生しなくなった → 再導入の障壁は消えている**

## 改善提案

---

### 🔴 今日中

#### ① 特定商取引法ページの決済表記を修正
**問題：** クレカ廃止後も「VISA/MASTER/DINERS/JCB/AMEX・ペイディ」の記載が残っている。虚偽表示として特定商取引法上のリスクがある。

**対応：** `/view/contract` の支払い方法欄から上記記載を削除し「銀行振込・掛け払い（法人審査制）」に修正。

---

### 🟠 今月中

#### ② クレジットカード決済の再導入【最重要・コンバージョン直結】
**背景：** 廃止理由だった「在庫切れ→返金の煩雑さ」が57mmの生産能力拡張で解消済み。

**対応：**
1. メイクショップの決済設定でクレカを再有効化（VISA/MASTER/JCB/AMEX/DINERS）
2. ショッピングガイドと特定商取引法ページを正式更新
3. 超大ロット（目安：10万個以上）受注時のみ「別途納期確認」を商品ページに追記

**競合状況：** バッジマンネット・缶バッジEXPRESS・ラッキープリント・パーツ販売ドットコム、全社クレカ対応。振込のみは法人の担当者にも負担（経費精算・経理処理の煩雑さ）。

#### ③ 価格表示の改善（商品名変更）【コスト0】
**問題：** 57mmカテゴリに「同じ2,000個、異なる価格」の商品が4つ並んでいる。実態はロット階段（1〜4ロット/5〜9ロット/10〜19ロット/20ロット以上）だが顧客に全く伝わっていない。

**メイクショップ制約：** プルダウンで単価変動する機能は50万円〜のカスタマイズが必要 → やらない。

**無料でできる対応：**
- 商品名にロット範囲を明記する（例：「【57mm】缶バッジパーツ ✕ 1〜4ロット（2,000〜8,000個）@19.11円」）
- 各商品ページの説明文に全ロット価格表を掲載
- カテゴリページ冒頭に「まとめ買いガイド」を設置

まず57mmから試して、他サイズへ展開。

### 🟡 3ヶ月以内

#### ④ EC内に「掛け払い」注文フローを整備【EC比率 30%→60% の鍵】
**問題：** メール注文書が多い主因は「社内の購買フロー上、先払いができない」こと。

**解決策：** 審査済み法人顧客がECサイト上で「掛け払い」を選択できる仕組みを作る。

```
改善後のフロー：
  法人顧客 → ECで注文 → 支払い方法：掛け払い選択
  → 注文確認メールが自動発行（社内の注文記録として使える）
  → 月末に三幸商事から請求書発行（従来通り）
```

**メイクショップでの実装：**
- 審査済み法人を「法人会員グループ」として登録
- その会員グループのみ「掛け払い」支払いオプションが表示される設定
- 標準機能（会員グループ別支払い方法出し分け）で対応可能なはず

**注意：** 主要顧客数社に「EC注文確認メールで社内承認が通るか」を事前確認してから展開する。自社フォーマット注文書が必要な顧客（約10%）は引き続きメール対応。

#### ⑤ 「業務用」「大量注文」向けコンテンツページの新設【SEO】
**問題：** 「缶バッジパーツ 業務用」「缶バッジ マシン 購入」でGoogle圏外。法人の購買担当が最初に検索するキーワードで負けている。

**対応：**
- 「業務用缶バッジパーツ 大量注文ガイド」専用ページを新設
- 国産・三幸グループ製造・安定供給・掛け払い対応を前面訴求
- 競合（canbadge-express等）が上位を取っているが、自社の強みで十分勝てる

#### ⑥ ブログをサブドメインからサブディレクトリへ移行【SEO】
**問題：** ブログが `up.badge-koi.com/blog/` のサブドメイン運営のため、ブログ記事のSEO効果がECサイト本体に蓄積されない。

**対応：** `www.badge-koi.com/blog/` に移行（301リダイレクト必須）。up.badge-koi.comの管理者と要調整。

### 🔵 6ヶ月〜（89期以降）

#### ⑦ schema.org 構造化データの実装
商品ページにProductスキーマを実装。Googleの検索結果に価格・在庫がリッチ表示され、クリック率向上。

#### ⑧ BtoBコンテンツメディアの強化
バッジマンネットが「ビジネスのための缶バッジ活用術」で先行。badge-koiは「国産ブリキ・カラーパーツの独自強み」「大量調達ノウハウ」をテーマにBtoB向け記事を月4本体制に引き上げる。

ターゲットキーワード例：
- 「缶バッジパーツ 業務用 大量注文」
- 「カラー缶バッジパーツ 種類・使い方」（競合ゼロの差別化コンテンツ）
- 「缶バッジパーツ 国産 輸入品 違い」

#### ⑨ Yahoo!ショッピング出店
競合（ラッキープリント）がYahoo!経由でPayPayポイント還元を活用して集客。新規法人顧客へのリーチ拡大。

#### ⑩ 英語ページ新設（89期海外展開連動）
主力商品（57mm・76mm・100mm）の英語ページ。台湾・韓国・北米向け缶バッジパーツ需要。競合ではバッジマンネットのみが対応。

## やらないこと（理由付き）

| 項目 | やらない理由 |
|------|------------|
| 小ロット対応（500個以下） | ターゲット外・現場負荷増大 |
| NPかけ払い等の外部掛け払いASP | 手数料が高コストで断念済み。自社審査で対応 |
| プルダウン単価変動機能 | メイクショップカスタマイズ50万円〜。費用対効果が合わない |
| フィルムクロスセル | 問い合わせがほぼ発生していない。低優先度 |
| メール文面注文・自社フォーマット注文書のEC移行強制 | 顧客にとってメールの方が楽な層は無理に変えない |
