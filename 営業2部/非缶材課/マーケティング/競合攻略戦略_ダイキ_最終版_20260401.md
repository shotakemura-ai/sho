# 缶バッジ競合攻略戦略 — 株式会社ダイキ
**作成日：2026年4月1日　　三幸商事株式会社 経営企画室**
**目標：缶バッジ売上 5.5億円 → 10億円（ダイキのパーツ・マシン事業を上回る）**

---

| 指標 | 三幸商事グループ | 株式会社ダイキ |
|------|----------------|--------------|
| 缶バッジ関連売上 | **5.5億円** | **15.7億円**（缶バッジ以外含む） |
| 事業構成 | パーツ90% / マシン10% | OEM完成バッジ＋パーツ＋マシン |
| 純利益率 | — | **16.05%**（¥2.52億） |
| 従業員 | — | **33名**（一人当たり47.5百万円） |
| 設立 | — | 1967年（57年） |
| 主要顧客 | funbox×CAN-缶、ベック、サザビー | Avex（12.67%）、KADOKAWA（12.2%） |
| 差別化軸 | カラーパーツ**国内唯一**・国産・SEO1位 | 57年の関係資本・エンタメ直取引 |
| OEM完成バッジ | **なし** | あり（主力） |

---

## 2. ダイキのビジネス構造と急所

```
【ダイキの収益レイヤー】

Layer A: OEM完成バッジ製造（Avex・KADOKAWA等向け）
         → 上位2社で売上の約25%（≒3.9億円）
         → 三幸商事は現在この市場に不在

Layer B: パーツ・マシン販売（缶バッジを作る企業向け）
         → 売上の残り75%（推定≒11〜12億円）
         → 三幸商事（5.5億）と直接競合する主戦場
```

**ダイキの急所 3点：**

| 急所 | 内容 | 三幸商事の優位 |
|------|------|--------------|
| ① ファブレス型 | 外注比率高（大王製作所31%・東洋品26%）。33名で品質管理に限界 | 三幸プロダクツが自社製造。品質・供給安定性で上回れる |
| ② カラーパーツ欠如 | ダイキはカラーパーツを販売していない（推定） | 三幸商事が国内唯一。カラーパーツを要望するバイヤーは三幸商事しか選べない |
| ③ 小口対応の非効率 | 顧客20社超だが、主力はKADOKAWA・Avex等の大口。中小顧客の対応優先度が低い | badge-koi.comで中小法人を効率的にカバー済み。ここを拡大するだけでよい |

## 3. 攻略戦略 3本柱

### 柱1【今すぐ・主戦場】パーツ市場でダイキの顧客を直接奪う

**ターゲット：ダイキからパーツを買っている中小グッズメーカー・MD会社**

ダイキのパーツ顧客 = 自社でプレスマシンを持ち、缶バッジを製造している法人。
これらの企業は現在badge-koi.comを使っていないか、ダイキと並行利用している。

**具体的アクション（優先順）：**

| # | アクション | 費用 | 期限 |
|---|-----------|------|------|
| ① | MD知人3人に個別訪問。「ダイキからパーツを買っているお客さんを紹介してほしい」と依頼。切り替えではなく**試しに1回使ってほしい**と伝える | 交通費・接待¥3万 | 今月中 |
| ② | HelfeCloudでダイキの顧客企業（販売先リスト下位）を抽出し、缶バッジパーツ購入可能性のある企業にbadge-koi.comへ誘導するアプローチメール送付 | 社内工数のみ | 来月中 |
| ③ | ダイキ競合向け刺さる訴求文作成：**「カラーパーツは弊社が国内唯一。ダイキにはありません」**を前面に出したLP（badge-koi.com内）を新設 | 社内制作¥5万 | 今月中 |
| ④ | 新規法人顧客への初回限定：カラーパーツ無料サンプルキット（57mm×200個 相当）を送付。ダイキへの乗り換えコストを実感させる | 材料費¥2万/件×10社 | 3ヶ月以内 |
| ⑤ | badge-koi.com クレジットカード決済再導入（設定変更のみ）。法人の経費精算負担を除去し、ダイキと比べた「使いやすさ」で差をつける | **¥0** | **今週中** |

### 柱2【3〜6ヶ月・波及効果大】マシン販売強化でパーツリピーターを増殖させる

**論理：マシンを1台売る = パーツ顧客を生涯確保する**

缶バッジマシンを三幸商事から買った企業は、三幸商事パーツを使い続ける。
マシン販売を現在の10%（0.55億）→20%（1.1億）に倍増させるだけで、
パーツのリピート売上が3〜5年にわたって積み上がる。

**具体的アクション：**

| # | アクション | 費用 | 期限 |
|---|-----------|------|------|
| ① | badge-koi.comに「缶バッジマシン」専用ランディングページ新設。現在「缶バッジマシン 購入」「缶バッジマシン 業務用」でGoogle圏外 → SEO1位を狙う | 社内制作¥5万 | 3ヶ月以内 |
| ② | **マシン購入後パーツ定期便プログラム**の設計：購入後6ヶ月間、月1回指定パーツが自動で届く（解約自由）。初回パーツ無料。ダイキにはないサービスで差別化 | 設計費¥0・初回原価のみ | 3ヶ月以内 |
| ③ | MD知人3人に「缶バッジビジネスを始めたいお客さん」を紹介してもらう。マシンとパーツのセット提案（マシン原価割引＋パーツで回収） | 紹介謝礼¥1万/件 | 来月中 |

---

### 柱3【12〜24ヶ月・構造変革】OEM完成バッジ市場への参入

**論理：ダイキの最大の収益源（Avex・KADOKAWA向けOEM）に直接参入する**

三幸プロダクツはプレス加工技術を持っている。技術的にはOEM完成バッジ製造は可能。
MDコネクション2〜3人を使ってパイロット案件を1件取れば、ダイキのOEM顧客への参入経路が開く。

**具体的アクション（シーケンス）：**

```
Step 1（今から6ヶ月）:
  MD知人3人に個別訪問。
  「1案件だけ試作無料でやらせてほしい」と提案。
  ダイキが担当していない案件から始める（競合を避けた実績作り）。
  → 費用：試作材料費¥10万以内

Step 2（6〜12ヶ月）:
  試作実績を携えてMD会社正式取引へ。
  「ダイキと並行した第2サプライヤー」として登録してもらう。
  ダイキに品質事故・納期遅延が起きた際に最初に呼ばれるポジションを取る。
  → 費用：接待・交通¥10万

Step 3（12〜24ヶ月）:
  実績2〜3案件を持って、Avex・KADOKAWA等エンタメ系へのアプローチ開始。
  ダイキとの差別化軸：国産三幸プロダクツ製造、大手商社グループの信用、BCP対応（供給継続保証）。
```

## 結論

> **「ダイキには正面攻撃せず、カラーパーツという武器でパーツ市場の顧客を一社ずつ引き剥がし、マシン販売でリピーターに育て、MDコネクションでOEM市場の入口を開ける」**

ダイキの最大の強みは57年の関係資本。価格でも品質でも短期では崩せない。
しかしカラーパーツ（国内唯一）という代替不可能な武器と、
badge-koi.comのSEO優位（パーツ通販Google1位）がある限り、
¥100万の予算でも3年以内に缶バッジ事業を倍増できる。
