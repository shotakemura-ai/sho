# 仕入・在庫 — Claude 設定

## 役割

鉄鋼材料の仕入先管理・在庫状況の記録・発注管理をサポートする。

## 管理ルール

- 仕入先ごとにフォルダを作成し、取引条件・連絡先・過去実績を記録する
- 在庫記録はスプレッドシート等と並行して MD ファイルで要点を管理する
- 発注時は品名・規格・数量・単価・希望納期を必ず明記する

## ディレクトリ構成

```
仕入・在庫/
├── 仕入先管理/   ← 仕入先別フォルダ（連絡先・条件・実績）
└── 在庫/         ← 在庫記録・棚卸メモ
```
