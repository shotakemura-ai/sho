# 鉄鋼・金属市場と世界経済動向に関する事業討議

![PLAUD NOTE](permanent/19abe18b80a3768d5c13eadda7dbb4c7/summary_poster/card_20260225052129-v2@edb4d58ad5fdeef1f22d62-1_20260225052432_da9cf3cb.png)## 鉄鋼・金属市場の現状分析
- ニッケルの供給・価格動向
  - ニッケルラインは3年前に拡張され、年間20万トン生産できる体制に。これにより市場が荒れた可能性が示唆された（20万トン、3年前）。
  - 価格の記録として「ニッケルは3800ドル」「3月から4月は1800ドル」との報告が併記され、期間や指標の違いにより大きな変動幅が確認される（数値は発言通り）。
  - 月産はニッケルが「ずっと振る」と表現され、月次の安定性に乏しい状況が示唆される。

- TCCと中国材の価格環境
  - TCCは中国材（発言上「中国治療」）に押されている。中国材が「1300ドル」から「いきなり3000ドル」に上がっているとの指摘があり、急騰局面での競争圧力が強い。
  - 「全体の数字50%」「コーナーだと50%」という比率言及があり、市場や製品カテゴリで半数規模の影響・構成比が示される。

- ブリキ（錫めっき鋼板）の国内需給・稼働
  - 去年、日本向けブリキは「来てない」「東京にゼロだった」との需要減の報告。
  - 国内生産・稼働の低下が著しく、「日本全体のブリキの1年の使用は50万トンいっていない」「150万トンが今40万トン」「今50万トンもいってない」と複数の水準が提示される過程で長期縮小が強調された。
  - 長期推移として「昔は180万トン台」「30年前は約120万トン」「100万トン切るか」まで低下し、現状は50万トン未満（数値は発言の通り時期別に併記）。
  - 稼働設備の例示：「ディーノナンバー2、ETLなど、ワンラインで3万トンの巨大ライン」だが、実際は「7000人しかやっていない」と稼働実数の少なさが指摘。
  - 国内メーカーの方針：「日本のブリキの稼働率は減らしている」「日鉄さんは輸出の方がはるかに多い」「国内販売シェアも3割とか」。輸出増加が確認される。

- 価格・取引実務の具体事例
  - 「東洋パックス向けの凍結剤」は「縦に2円だけ負けてくれる」「84円」。一方で「そんな価格は出ない」「他社からも出ない」との現場感が示され、提示価格と市場受容の乖離がある。
  - 「パックさんは103円」「別ループでは120何歩」で出していたとの併記があり、サプライヤー間の足並みと提示条件の差が語られた。
  - 「去年の夏に1万円下げる」との価格調整の事例がある。

- 需要構造・市場縮小の背景
  - 缶需要の縮小：「東洋製缶は多数」「医療缶が激減」「北海製缶が新業ライン撤退（去年）」。
  - 副次市場の縮小：「カレンダークリップは最近だいぶ減った」「カレンダー自体も減っている」。
  - 輸出は増えているが「お金ならない」収益性の低さが問題視される。

- 韓国市場との対比
  - 韓国ブリキ市場は過去「1日50万トン超」から現在「27、8万トン」へ縮小。
  - 主需要「コアロン（缶底板）」、エアゾール缶の生産が歴史的に多く、日本も輸入してきた。
  - 韓国人口「5500万」。一人当たりブリキ使用量は日本より多いとの評価がある。

- 原材料価格とコスト圧力
  - スズは「5万ドル超」から「今は4万5千、6千くらい」「ダブルくらいに」上昇。半導体部材のはんだ用途などで需要が高く、供給不足感が強い。
  - ホットコイルの見通しは「落ち着く」との観測。韓国では中国材・日本材に反ダンピング、「日本は35%」「ヒュンダイ製鉄が掛けた」。
  - 電気料金は「50%以上上がっている」。以前は「日本の半分、3分の1」だったが、産業用で特に上昇。
  - 東南アジアのブリキ価格はUSドルベースで「ホット500」「ブリキが800〜900ドル前後」との水準感が提示された。
  - 低板厚の仕様言及：「2CRの0.14、0.15、0.16のロウゲージ」。

- 価格下落余地に関する認識
  - 「値段がこれ以上上がることはない」「株価がこれ以上上がることはない」という見方がある一方で、コスト上昇下で「これ以上下がることは絶対ない」「下がったらみんな死んでしまう」「工場止めてた方が」と価格下方硬直性の強調が複数回示された。

- 在庫・オペレーション
  - 在庫「8ヶ月分」があるとの具体的情報が共有された。

## EV市場と電池技術の動向
- 市場トレンドの変化
  - 欧州メーカーのEV戦略は「トーンダウン」。政策・市場要因により伸びが鈍化。
  - アメリカではトランプ元大統領の「EVはいらない」との発言に象徴される政治的逆風が指摘され、欧州にも波及との見方。
  - 中国は「生産大量」「在庫が多すぎる」ことが需給悪化の一因。
  - 欧州の目標時期は「2028年に狙ったものを32年とか」「5年ぐらい延ばしている」との期日の延伸が語られた。
  - 足元の見通しは「やっと今になってまだEVがちょっと上向いていくんじゃないかな」という回復期待の声もある（評価表現は発言通り）。

- ハイブリッドの再評価と電池技術シフト
  - 「今はハイブリッド、トヨタさんのハイブリッドの時代」「結構売れてます」との需要増。
  - 技術転換：「ニッケル水素電池」から「リチウムイオン電池のハイブリッド」へのシフトが進行。
  - 材料影響：「ニッケルメッキ鋼板」の需要は、リチウム系への移行で「いらない」ケースが増え、増勢は見込めないとの見解。
  - トヨタの仕様：「トヨタさんは（リチウム系で）使ってない」「核はアルミ」「間じゃない」。ニッケルメッキ鋼ではなくアルミ主体の構成が言及された。
  - パナソニックの丸形電池は「電池の間をやるシステム」で「ニッケルメッキ鋼」を使用する事例が示された。

- メーカー別戦略の含意
  - BYDは日本では「ほとんど知らない」メーカーとの受け止めが共有。
  - イーロン・マスクのEV観：「安物はBYDに任せる。ハイクラスだけテスラ」との発言紹介。「EVから撤退した資金をロボットへ」という資金配分の話題が語られた。

## 世界経済とテクノロジーの潮流
- AIの進展と労働市場リスク
  - 「AIが発展すればするほど人間はいらなくなる」「単純作業は代替され失業者が増える」との懸念が提示。
  - 経済連鎖リスク：「失業率上昇→不動産・消費低迷→借入返済困難→金融不安」という大きな裏側リスクが指摘された。

- 企業のAI・ロボット導入
  - 韓国大手では「スタッフがほとんどAIでチェックピッチなど全部企画」に言及。
  - ヒュンダイは「ロボットに投資」。ただし「労働組合とものすごく揉めている」状況。工場のロボット化に対し組合は反対、企業は導入せざるを得ない立場。

- 中国のロボット・宇宙技術
  - 中国のロボット技術は「人間と同動作のパフォーマンス（YouTube参照）」のレベルまで進化。「世界でナンバーワンかそれに近い」「政府が莫大な資金投入」「技術さえあればお金があり、毎日何百社もの新会社」。展示会で「信じられないくらいの実力」も示される。
  - 宇宙分野は「SpaceXと同様に評価」。アメリカには及ばないが「世界ナンバーワン、ナンバーツーのレベル」という見方。

- イーロン・マスクの動向
  - 「人間ではないような行動や発言が多い」との評。EVの安価帯をBYDへ委ね、テスラはハイクラスへ、資金をロボットに注入との話題が共有された。
  - ロボット技術の軍事転用可能性が将来リスクとして示される。

## 各国市場の比較分析と将来展望
- 日本のブリキ・製鉄需給と輸出
  - 長期需給縮小の定量情報：「日本全体のブリキの1年の使用は50万トンいっていない」「150万トンが今40万トン」「今50万トンもいってない」「昔は180万トン台」「30年前は約120万トン」「100万トン切るか」。
  - 設備と稼働：「ラインはワンライン3万トン級」「実際は7000人しかやっていない」「稼働率は低い」。
  - 輸出は増加。「日鉄は輸出の方がはるかに多い」「国内販売シェア3割とか」。品質チェックの扱いに差があるとの現場感が示された。

- 韓国市場・原価環境・価格政策
  - 韓国ブリキは「1日50万トン超」から「今は27万〜28万トン」へ。主需要は「コアロン」。エアゾール缶の生産が多く、日本は長年輸入。
  - 韓国の電気料金は「50%以上上がっている」。かつては「日本の半分、3分の1」。産業用で特に上昇。
  - 反ダンピング：ホットコイルの中国材・日本材に対して実施。「日本は35%」「ヒュンダイ製鉄が掛けた」。
  - 価格観：「東南アジアのブリキは900ドル前後」「ホットは500ドル」との相場観。「ブリキ価格はこれ以上上がらない」「景気回復前で上昇見込みなし」とPOSCO役員の見解が共有。

- 価格ボトム認識と外部リスク
  - 「ブリキ価格がこれ以上下がることはない」「下がればみんな死んでしまう」との強い下方硬直認識。
  - 中東情勢がブリキ価格に影響を与え、「戦争が起きれば値段が上がる可能性」がリスクとして示され、需要も中東影響を大きく受けるとの見方。

- マクロ環境：インフレ・為替・株式・企業業績
  - 日本のインフレは「3年前から」。食料品が最も上昇。「食料品の消費税は8%」で「ゼロにする動き」がある。
  - 為替：「1ドル155円」。政府が円安誘導との認識。円安は輸出企業に有利でインフレを招く。
  - ガソリン：国が税を取りやめたため、円安での上昇分が相殺され下がったとの報告。
  - 株式：日本の株価は「過去最高水準」「日経平均株価は5万7千～8千円」。韓国も「過去最高」「1月、2月で50%上がった」「4千が6千5百を超えた」。
  - 政策：韓国は「新しい李大統領」が不動産から株・金融へ方針転換、「自社株買いを強制」し株価急騰。
  - 産業：韓国は「半導体と一部AI関連」で株価伸長。軍事産業は「3年で10倍」増、日本も「川崎重工、宇都宮重工」が同様の動き。
  - 企業業績：日本の一部上場企業（プライム市場1000社）は「5年連続で経常利益プラス」「今50兆円」。インフレと輸出の影響が大きい。大手企業の業績は良好で株価も過去最高。

- サムスン・投資・観光と物価
  - サムスンは半導体で「今年だけで200兆ウォン（20兆円）の利益」。ピョンテックで半導体工場を建設中、新規投資が進行。
  - 観光：日本から韓国へ中国観光客が流れており、明洞は日本の観光客が多い。日本は中国観光客来訪でホテル価格が上がり「出張に行けなくなる」との影響感。韓国のビジネスホテルは値上がり。日本のホテルは「昨年比で2倍、3倍」「同ホテルが2万円→4万円」の事例。
